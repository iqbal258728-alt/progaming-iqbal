Pengertian CSS

(CSS) adalah bahasa style sheet yang digunakan untuk mengatur tampilan, tata letak, warna, font, dan nuansa visual dokumen HTML atau XML, membuat halaman web menjadi responsif dan menarik. CSS memisahkan konten dari presentasi visual, memungkinkan konsistensi gaya di berbagai halaman web.
